# DIY-PS503A-Power-Supply
DIY standalone version of the Tektronix PS503A dual tracking power supply

Lab Power Supply with dual tracking over-voltage protection.
- Output voltage +/- 0..20V
- Current limiting 0..300mA or 0..600mA
- Output overvoltage protection > 24V
- Tracking overvoltage protection with a delta of 1.4V
- Auto-ranging display with voltmeter
- Dual tracking mode with the same voltages or a percentage change
- No fan
- 230V input with R-Core transformer and input filter
